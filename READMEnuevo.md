Proyecto Calculadora Rural, para calcular  la producción que le dan sus animales y también cuánto dinero podría generar. Muchas veces esos cálculos se hacen a mano o mentalmente y se pueden cometer errores. Con esta página el usuario solo pone el tipo de animal, la cantidad, la producción y el precio, y el sistema ya calcula la producción diaria, mensual y los ingresos.
La  página estaba funcionando con Python así que eliminé este archivo y ahora toda la lógica está en un archivo js, añadí un par de ciclos para no repetir los cálculos y también agrupé el código en funciones para que sea más profesional y organizado.
Los comandos en la terminal que utilicé fueron:
: git add .
>> git commit -m "mejora de proyecto"
>> git push origin main
Para actualizar el proyecto en git 
correr el programa a través de la terminal con start index.html
El de dir porque quería ver los archivos del proyecto
Los atajos que aplique fueron Ctrl +shift+ f para buscar funciones en los archivos, comentar y descomentar una línea de código Ctrl + / , el Ctrl+ñ  abrir la terminal y  Ctrl+ f para buscar dentro del archivo que me encuentro.
Se puede correr por medio de la terminal integrada con start index.html o por medio de start debuggin.
